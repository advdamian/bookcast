# Bookcast

Bookcast este o platformă care găzduiește **Ziarul Buzz**, interviuri și podcasturi cu personalități inspiraționale, precum și creații ale elevilor.

## tehnologii folosite:
* **HTML5**
* **Tailwind CSS**
* **Vanilla JavaScript** 
* **Brevo & Formspree** (Pentru managementul newsletter-ului și al formularelor de contact)

Proiectul foloseste clean urls si este construit modular pentru adaugarea, stergerea sau modificarea cu usurinta a oricarei parti.
